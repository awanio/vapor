Vapor API Configuration Example

To override the default API endpoints, add the following script to your index.html file before loading the app:

<script>
    // Configure API endpoints before the app loads
    window.VAPOR_CONFIG = {
        API_BASE_URL: 'http://your-api-host:8080',
        WS_BASE_URL: 'ws://your-api-host:8080',
        API_VERSION: '/api/v1'  // Optional, defaults to '/api/v1'
    };
</script>

Default Configuration

By default, the app uses:

  • API URL: http://103.179.254.248:8080
  • WebSocket URL: ws://103.179.254.248:8080
  • API Version: /api/v1

Example for Different Environments

Production

window.VAPOR_CONFIG = {
    API_BASE_URL: 'https://api.example.com',
    WS_BASE_URL: 'wss://api.example.com'
};

Local Development

window.VAPOR_CONFIG = {
    API_BASE_URL: 'http://localhost:8080',
    WS_BASE_URL: 'ws://localhost:8080'
};

Custom Port

window.VAPOR_CONFIG = {
    API_BASE_URL: 'http://192.168.1.100:3000',
    WS_BASE_URL: 'ws://192.168.1.100:3000'
};
